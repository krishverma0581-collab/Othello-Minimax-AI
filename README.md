# Othello with Minimax AI

A desktop-based Othello (Reversi) game developed using Python and Tkinter.

## Features
- Human vs AI gameplay
- Minimax algorithm for AI decision making
- Valid move highlighting
- Score tracking
- Restart game functionality
- Game over detection and winner announcement
- Interactive GUI using Tkinter

## Technologies Used
- Python
- Tkinter
- Minimax Algorithm
- Object-Oriented Programming concepts

## Project Structure
- othello.py -> Game logic and Minimax algorithm
- gui.py -> Tkinter graphical interface

## How to Run
1. Install Python
2. Open terminal
3. Run:

python gui.py
